 <p>
  【大纪元2021年06月19日讯】（大纪元记者王祥报导）美国媒体报导说，传
  <span class="s1">
   <ok href="https://www.epochtimes.com/gb/tag/%E4%B8%AD%E5%85%B1.html">
    中共
   </ok>
   高层正在讨论到
  </span>
  <span class="s2">
   2025
  </span>
  <span class="s1">
   年完全取消
   <ok href="https://www.epochtimes.com/gb/tag/%E7%94%9F%E8%82%B2%E9%99%90%E5%88%B6%E6%94%BF%E7%AD%96.html">
    生育限制政策
   </ok>
   的可能。
  </span>
 </p>
 <p class="p3">
  <span class="s1">
   《华尔街日报》周五（6月18日）引述消息人士的话说，
   <ok href="https://www.epochtimes.com/gb/tag/%E4%B8%AD%E5%85%B1.html">
    中共
   </ok>
   官员正着手制定计划、进一步放宽生育限制，并逐渐过渡到鼓励生育的政策。
  </span>
 </p>
 <p class="p3">
  <span class="s1">
   报导说，这反映出随着经济增长放缓和中国人口结构走向老龄化，中国（中共）政府感到越来越大的紧迫性。
  </span>
 </p>
 <p class="p3">
  <span class="s1">
   其中一位消息人士称，中共的政策制定者正在讨论到
  </span>
  <span class="s3">
   2025
  </span>
  <span class="s1">
   年完全取消生育限制的可能，出生率最低的东北省份可能会率先取消生育限制，然后推广至全国。
  </span>
 </p>
 <p class="p3">
  <span class="s1">
   中国东北三省在
  </span>
  <span class="s3">
   1950
  </span>
  <span class="s1">
   年代曾是中国的老工业基地，但在
  </span>
  <span class="s3">
   1990
  </span>
  <span class="s1">
   年代受市场经济转变的严重打击后，这些省部分地区的出生率在全国排在倒数之列。最新人口普查数据显示，在过去十年里，东北三省的人口下降了
  </span>
  <span class="s3">
   1.2%
  </span>
  <span class="s1">
   。
  </span>
 </p>
 <p class="p3">
  <span class="s1">
   人口和经济学者表示，需要采取积极行动来阻止中国即将出现的出生人口下滑局面。英国医学
  </span>
  <span class="s4">
   杂志《柳叶刀》
  </span>
  <span class="s1">
   （
  </span>
  <span class="s3">
   The Lancet
  </span>
  <span class="s1">
   ）去年
   <span class="s5">
    发表的一份预测
   </span>
  </span>
  <span class="s6">
   说
  </span>
  <span class="s1">
   ，到本世纪末，中国人口数量预计将从目前的
  </span>
  <span class="s3">
   14.1
  </span>
  <span class="s1">
   亿人下降到约
  </span>
  <span class="s3">
   7.3
  </span>
  <span class="s1">
   亿人。
  </span>
 </p>
 <p class="p3">
  <span class="s1">
   虽然人口数量变少不等于是坏事，但畸形
   <ok href="https://www.epochtimes.com/gb/tag/%E8%AE%A1%E5%88%92%E7%94%9F%E8%82%B2.html">
    计划生育
   </ok>
   政策给人口结构在未来埋下的大弊端已陆续开始显现——中国
   <span class="s5">
    没有足够的劳动年龄人口
   </span>
   来支持增长的退休人口——可能会在接下来的几十年里拖累中国经济。
  </span>
 </p>
 <p class="p3">
  <span class="s1">
   中共政府一直不愿意彻底放弃生育控制政策。华日引述一位知情人士的话说，中共领导人担心，完全放开生育限制可能会鼓励农村地区贫困家庭生更多的娃，加剧部分地区的经济问题。
  </span>
 </p>
 <p class="p3">
  <span class="s1">
   自
  </span>
  <span class="s3">
   1980
  </span>
  <span class="s1">
   年以来，
   <ok href="https://www.epochtimes.com/gb/tag/%E8%AE%A1%E5%88%92%E7%94%9F%E8%82%B2.html">
    计划生育
   </ok>
   一直是中共的一项重要政策。许多家庭因为超生被罚款，还有家庭被强迫绝育和堕胎。
  </span>
 </p>
 <p class="p3">
  <span class="s3">
   2013
  </span>
  <span class="s1">
   年，中共政府称至少一方是独生子女的家庭可以生育两个孩子，不受惩罚。
  </span>
 </p>
 <p class="p3">
  <span class="s1">
   2016年，中共政府宣布取消了独生子女政策，当年出生人口曾上升，但随后年份再次下降。
  </span>
 </p>
 <p class="p3">
  <span class="s1">
   2021年5月，中共宣布进一步放开三胎政策，每十年一次的
   <span class="s5">
    人口普查
   </span>
   显示，中国这个全球人口最多的国家正处于历史性的人口衰退边缘。
  </span>
 </p>
 <p class="p3">
  <span class="s1">
   人口学家对其能否
   <span class="s5">
    带来长久的改变
   </span>
   持悲观态度。
  </span>
 </p>
 <p class="p3">
  <span class="s1">
   北京大学人口研究所教授穆光宗表示，中共政府需要承认，自主生育是一项基本人权。他称，在实施计划生育政策的情况下，无论允许生几个孩子，中国人自主生育权利的事情并没有解决。
  </span>
 </p>
