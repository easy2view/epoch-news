 <p>
  【大纪元2021年06月23日讯】（大纪元记者林燕综合报导）
  <span class="s3">
   《华尔街日报》周三（6月23日）引述
  </span>
  <span class="s1">
   知情人士的消息说，
   <ok href="https://www.epochtimes.com/gb/tag/%E9%A9%AC%E4%BA%91.html">
    马云
   </ok>
   旗下的
   <ok href="https://www.epochtimes.com/gb/tag/%E8%9A%82%E8%9A%81%E9%9B%86%E5%9B%A2.html">
    蚂蚁集团
   </ok>
   正在与中共国有企业洽谈，考虑创建一家信用评分公司，并贡献出蚂蚁的专有消费者数据、便于监管机构管辖。
  </span>
 </p>
 <p class="p5">
  <span class="s1">
   报导称，新实体可能最快在今年第三季度成立，
   <ok href="https://www.epochtimes.com/gb/tag/%E8%9A%82%E8%9A%81%E9%9B%86%E5%9B%A2.html">
    蚂蚁集团
   </ok>
   可能会因此失去其掌握的中国民众金融消费习惯数据的部分控制权。
  </span>
 </p>
 <p class="p3">
  <span class="s1">
   有超过
  </span>
  <span class="s4">
   10
  </span>
  <span class="s1">
   亿人使用这家金融科技巨头的支付宝程序来消费、借贷或投资，蚂蚁集团收集和使用的这些用户信息是该公司
  </span>
  <span class="s5">
   获得成功
  </span>
  <span class="s1">
   的秘诀。
  </span>
 </p>
 <p class="p5">
  <span class="s1">
   报导说，亿万富翁
   <ok href="https://www.epochtimes.com/gb/tag/%E9%A9%AC%E4%BA%91.html">
    马云
   </ok>
   控制的蚂蚁集团与中共国有企业之间的谈判很可能催生一家合资公司，该合资公司将获得许可成为一家信用评分公司。
  </span>
 </p>
 <p class="p5">
  <span class="s1">
   据知情人士称，蚂蚁集团和监管机构也一直在讨论该合资公司是否应该由蚂蚁集团或国有企业来运营和控制。
  </span>
 </p>
 <p class="p5">
  <span class="s1">
   据一些了解谈判内情的人士透露，中共监管机构正在推动潜在国有股东在新实体中发挥更重要作用，以便对该实体的运营方式拥有更大发言权。潜在股东包括一家总部位于上海的金融集团。
  </span>
 </p>
 <p class="p5">
  <span class="s1">
   知情人士还表示，关于该新实体将收集哪些类别的数据以及由此产生的信用评分将如何匹配中共当局拟建的全国性数据库计划，也已展开磋商。
  </span>
 </p>
 <p class="p5">
  <span class="s1">
   知情人士称，上述讨论正在继续进行，尚未做出最终决定。
  </span>
 </p>
 <p class="p5">
  <span class="s1">
   蚂蚁集团的发言人不予置评。中共央行中国人民银行未回应置评请求，它们正监督指导蚂蚁集团进行
   <span class="s6">
    更全面的整改
   </span>
   。
  </span>
 </p>
 <p class="p5">
  <span class="s1">
   蚂蚁集团早在六年前就试图推出自己的全国性信用评分系统——芝麻贷，但在收到当局限制后作罢。
  </span>
 </p>
 <p class="p5">
  <span class="s1">
   中国至今还没有一个类似美国FICO的全国性信用评分系统。虽然
  </span>
  <span class="s1">
   人行旗下有一个信用参考中心——从各银行和其它金融机构收集的个人和公司的信用信息——但缺乏那些尚无资格获得传统银行贷款的人群信息。
  </span>
 </p>
 <p class="p5">
  <span class="s1">
   在过去十年来，蚂蚁集团等金融科技公司加大了对这部分空白记录人群的服务，并将收集到的个人信息保存在自己的系统中，并
  </span>
  <span class="s1">
   没有获得用户同意，拒绝跟当局分享其数据或将其数据上传其它金融机构可访问的中央存储库。
  </span>
 </p>
 <p class="p5">
  <span class="s1">
   但在蚂蚁集团去年上市被意外叫停，以及马云的
   <ok href="https://www.epochtimes.com/gb/tag/%E9%98%BF%E9%87%8C%E5%B7%B4%E5%B7%B4.html">
    阿里巴巴
   </ok>
   集团今年接连收到中共当局打击后，情况发生了变化。
  </span>
 </p>
 <p class="p5">
  <span class="s1">
   本月早些时候，蚂蚁集团成立了一家消费金融公司，其股东也包括国有和私营企业。这家公司将改变蚂蚁集团资助和提供部分短期贷款的方式。
  </span>
 </p>
 <p class="p6">
  <span class="s7">
   《华日》6月中旬报导说，
  </span>
  <span class="s8">
   中共已展开一场新权力游戏，力图控制中国科技公司手中的数据宝库。
  </span>
 </p>
 <p class="p7">
  <span class="s1">
   “中国（中共）领导人担心，中国的科技巨头可能利用他们的大量个人和企业数字记录，在一党专制国家建立替代性的权力中心。”报导说。
  </span>
 </p>
 <p class="p8">
  <span class="s1">
   报导还引述参与中共政策制定的消息人士的话说，中共领导人越来越意识到，应将私营部门积累的数据视为国家资产，然后根据国家的需要进行挖掘或限制。
  </span>
 </p>
 <p class="p7">
  <span class="s1">
   习近平已越来越倾向于主张加强数字控制。他将大数据列为中国经济的另一个基本要素，与土地、劳动力和资本并列。
  </span>
 </p>
 <p class="p7">
  <span class="s1">
   据熟悉内部讨论的人士称，习近平在私会上说：“谁控制了数据，谁就掌握了主动权。”
  </span>
 </p>
