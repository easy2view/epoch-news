 <p>
  【大纪元2021年06月19日讯】（大纪元记者林燕综合报导）
  <span class="s1">
   今年夏天的第一场大热浪已经席卷美国西部，影响到西部
  </span>
  <span class="s2">
   四千
  </span>
  <span class="s1">
   多万居民。至少有五个气象站，包括盐湖城和加州的棕榈泉的温度在本周达到了历史最高，虽然今年夏季才开始不久。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   拉斯维加斯的最高温度达到
  </span>
  <span class="s2">
   116
  </span>
  <span class="s1">
   度（46.7摄氏度），创下历史最高；凤凰城的温度为
  </span>
  <span class="s2">
   117
  </span>
  <span class="s1">
   度（47.2摄氏度），比该市之前的日纪录高出
  </span>
  <span class="s2">
   3
  </span>
  <span class="s1">
   度；加州的死亡谷显示最高温度是
  </span>
  <span class="s2">
   128
  </span>
  <span class="s1">
   度（53.3摄氏度）。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   拉斯维加斯的国家气象局在推特上呼吁居民留在室内、保持水分：
  </span>
  <span class="s2">
   “
  </span>
  <span class="s1">
   长时间的热浪可致命。
  </span>
  <span class="s2">
   ”
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   棕榈泉、凤凰城的医生们已警告说，人行道非常炙热，可能导致三度烧伤。对于老年人、无家可归者或患有心血管疾病和糖尿病等人群来说，极端高温天气可能是致命的。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   华盛顿州立大学气候科学家迪普提
  </span>
  <span class="s1">
   ‧辛格（
  </span>
  <span class="s2">
   Deepti Singh
  </span>
  <span class="s1">
   ）说，今年的第一场热浪在时间、强度和范围上都是史无前例；从来没有在夏天这么早的时候在这么大的区域内记录过如此严重的热浪情况。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   辛格说，再加上灾难性的干旱，农作物受损，水库水位已萎缩到历史最低点。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   现在尚不清楚炙热天气会持续到何时。预计本周末西南地区的各城市温度将达到
  </span>
  <span class="s2">
   115
  </span>
  <span class="s1">
   度或更高（46.1摄氏度）。新墨西哥州、犹他州、亚利桑那州和科罗拉多州都发出了严重警告，因为闪电和狂风可能会引发野火。
  </span>
 </p>
 <p class="p1">
  随着热浪而来的是大规模使用空调，西部电网供应变得更加紧张，德克萨斯州和加利福尼亚州的公用事业部门警告说，可能会断电。
 </p>
 <p class="p1">
  <span class="s1">
   政府建议民众就近避暑，比如：树荫、操场喷淋、政府大楼的空调大厅等。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   受热浪影响，夜间温度也很高。凤凰城过去两个晚上的温度都没有降到
  </span>
  <span class="s2">
   90
  </span>
  <span class="s1">
   度以下（32.2摄氏度）。到了早上，太阳感觉像是在烤人，皮肤几乎都被烤焦了。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   亚利桑那州马里科帕县有持续追踪高温天气对健康影响的记录，去年一年就有
  </span>
  <span class="s2">
   三百
  </span>
  <span class="s1">
   多人死于高温。自今年
  </span>
  <span class="s2">
   4
  </span>
  <span class="s1">
   月以来，该县已记录了
  </span>
  <span class="s2">
   3
  </span>
  <span class="s1">
   起与高温有关的死亡案例，并正在调查另外
  </span>
  <span class="s2">
   20
  </span>
  <span class="s1">
   起案例。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   该地区的政府机构和社区团体已将避难所和办公室变成紧急避暑中心。在凤凰城南部的非营利性组织圣文森特
  </span>
  ‧
  <span class="s1">
   德‧保罗校园（St. Vincent de Paul）里，餐厅被改造成临时住宿场所，安置了
  </span>
  <span class="s2">
   200
  </span>
  <span class="s1">
   张小床，入口处的冷藏箱里放着水、冰棍和冰块。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   在加利福尼亚的中央谷地，女性农场工人协会（
  </span>
  <span class="s2">
   Líderes Campesinas
  </span>
  <span class="s1">
   ）的志愿者开车从一个田地到另一个田地配发水，并提醒工人们注意高温危险。他们的建议是：注意休息，待在阴凉处，留意是否有中风迹象。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   美国西部一半以上的地区目前正经历着
  </span>
  <span class="s2">
   “
  </span>
  <span class="s1">
   极端
  </span>
  <span class="s2">
   ”
  </span>
  <span class="s1">
   或
  </span>
  <span class="s2">
   “
  </span>
  <span class="s1">
   特殊
  </span>
  <span class="s2">
   ”
  </span>
  <span class="s1">
   干旱，因广泛缺水，可能对农作物和牧场造成重大影响。犹他州和科罗拉多州山区的关键积雪比正常水平低
  </span>
  <span class="s2">
   30%
  </span>
  <span class="s1">
   至
  </span>
  <span class="s2">
   50%
  </span>
  <span class="s1">
   ，并在迅速融化。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   西部主要水库的水位已达到历史最低点。联邦政府已将对加州一些城市的用水配额削减了
  </span>
  <span class="s2">
   75%
  </span>
  <span class="s1">
   ，整个西南地区的农民都在担忧农作物歉收问题。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   此外，预防野火也是一个重点。高温和干旱使得植被、大片的森林变成潜在的野火燃料。另一方面，因为高温天气，亚利桑那州的消防官员表示，一些消防飞机无法安全飞行。
  </span>
 </p>
 <p class="p1">
  气候科学家辛格说，所有情况都表明，今年比去年更糟糕。他去年有几周时间都在躲避野火灾害。
 </p>
