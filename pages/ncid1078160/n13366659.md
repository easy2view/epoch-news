  <ok href="https://www.epochtimes.com/gb/tag/%E5%AF%B9%E8%AF%9D.html">
   对话
  </ok>
  。这些女孩的童言童语往往会让人发笑，这也难怪他的粉丝已经超过100万名。以下摘译他最近分享的一些有趣推文，以飨读者：

 </p>
 <p>
  （25）
  <br/>
  （去看牙医）
  <br/>
  我：“你会勇敢吗？”
  <br/>
  5岁
  <ok href="https://www.epochtimes.com/gb/tag/%E5%A5%B3%E5%84%BF.html">
   女儿
  </ok>
  ：“会，但当我需要尖叫时除外。”
 </p>
 <p>
  （26）
  <br/>
  我：“你的老师寄电子邮件跟我说，你有一次作业没交。”
  <br/>
  9岁女儿：“她昨晚寄给你的吗？”
  <br/>
  我：“是的。”
  <br/>
  9岁女儿：“她就不能休息一晚吗？”
 </p>
 <p>
  （27）
  <br/>
  7岁女儿：“我看起来有比较高吗？”
  <br/>
  我：“没有。”
  <br/>
  7岁女儿：（吃下第5块鸡块）“现在呢？”
 </p>
 <p>
  （28）
  <br/>
  7岁女儿：“当我结婚时，我一定要穿白纱吗？”
  <br/>
  我：“你可以穿你想穿的服装。”
  <br/>
  7岁女儿：“即便是忍者装？”
 </p>
 <p>
  （29）
  <br/>
  11岁女儿：“我需要20美元在学校里买另一样东西。”
  <br/>
  我：“你真会花钱。”
  <br/>
  11岁女儿：“我值得。”
 </p>
 <p>
  （30）
  <br/>
  5岁女儿：“我早餐要吃炒蛋。”
  <br/>
  我：“你刚才说你早餐要吃培根。”
  <br/>
  5岁女儿：“你应该知道我什么时候会改变心意。”
 </p>
 <p>
  （31）
  <br/>
  7岁女儿：“我看了你的YouTube频道。”
  <br/>
  我：“所以你喜欢我的影片？”
  <br/>
  7岁女儿：“不喜欢。”
 </p>
 <p>
  （32）
  <br/>
  7岁女儿：“你猜我在想什么。”
  <br/>
  我：“什么？”
  <br/>
  7岁女儿：“食物。”
  <br/>
  （她毕竟是我的孩子。）
 </p>
 <p>
  （33）
  <br/>
  我：“你忘了洗餐盘。”
  <br/>
  11岁女儿：“我没忘记。”
  <br/>
  我：“但你没洗餐盘。”
  <br/>
  11岁女儿：“但我没忘记。”
 </p>
 <p>
  （34）
  <br/>
  7岁女儿：“我们今天要做什么？”
  <br/>
  我：“要做对这个家有好处的所有趣事。”
  <br/>
  7岁女儿：“那是清理房屋的密码吗？”
  <br/>
  （她破解了密码。）
 </p>
 <p>
  （35）
  <br/>
  我：“把头发梳一梳。”
  <br/>
  9岁女儿：“我梳过了。”
  <br/>
  我：“再梳一次。”
  <br/>
  9岁女儿：“我的头发不给我第二次机会。”
 </p>
 <p>
  （36）
  <br/>
  我：“你为什么还没洗澡？”
  <br/>
  9岁女儿：“我很忙。”
  <br/>
  我：“忙什么？”
  <br/>
  9岁女儿：“忙着不洗澡。”
 </p>
