 <p>
  （13）
  <br/>
  5岁
  <ok href="https://www.epochtimes.com/gb/tag/%E5%A5%B3%E5%84%BF.html">
   女儿
  </ok>
  ：“我因为阅读书本而得到免费的披萨。”
  <br/>
  我：“真棒。”
  <br/>
  5岁女儿：“阅读能当工作吗？”
 </p>
 <p>
  （14）
  <br/>
  我：（吃光每个人餐盘里的剩余食物）
  <br/>
  9岁女儿：“哇。”
  <br/>
  我：“什么？”
  <br/>
  9岁女儿：“只要有你在，我们就不需要垃圾桶。”
 </p>
 <p>
  （15）
  <br/>
  我：“为什么这件衬衫在洗衣房里？你穿过了吗？”
  <br/>
  9岁女儿：“我刚从抽屉里拿出来。我不想再把它折叠回去。”
 </p>
 <p>
  （16）
  <br/>
  11岁女儿：（要去她朋友家过夜）
  <br/>
  我：“你希望我几点去接你？”
  <br/>
  11岁女儿：“别来（接我）。”
 </p>
 <p>
  （17）
  <br/>
  9岁女儿：“你准备要摧毁我们的夜晚吗？”
  <br/>
  我：“没有，我只是看看新的食谱。”
  <br/>
  9岁女儿：“一样啦。”
 </p>
 <p>
  （18）
  <br/>
  11岁女儿：“明天我们可以打扮成一个角色去上学。”
  <br/>
  我：“你想打扮成谁？”
  <br/>
  11岁女儿：“披上隐形斗篷的哈利波特（Harry Potter）。”
  <br/>
  我：“你（即便隐形）还是得去上学。”
 </p>
 <p>
  （19）
  <br/>
  7岁女儿：“我整理我的房间了。”
  <br/>
  我：“这让我印象深刻。你怎么这么乖？”
  <br/>
  7岁女儿：“妈妈（叫我做的）。”
 </p>
 <p>
  （20）
  <br/>
  我：“不要和你的妹妹们打架。”
  <br/>
  11岁女儿：“我们不是在打架。”
  <br/>
  我：“那你为什么吼叫？”
  <br/>
  11岁女儿：“那是我们说话的方式。”
 </p>
 <p>
  （21）
  <br/>
  9岁女儿：“我洗餐盘了。”
  <br/>
  我：“它们还是很脏。”
  <br/>
  9岁女儿：“我没说我洗得很干净。”
 </p>
 <p>
  （22）
  <br/>
  9岁女儿：“当我长大，我要待在家里，什么事也不做，就像你一样。”
  <br/>
  我：“我是在家里工作。”
  <br/>
  9岁女儿：“别毁了我的梦想。”
 </p>
 <p>
  （23）
  <br/>
  太太：“你老爸在做什么？”
  <br/>
  11岁女儿：“在浪费时间。”
  <br/>
  我（从隔壁房间高喊）：“我在看美式足球比赛。”
  <br/>
  11岁女儿：“那就是我说的意思。”
 </p>
 <p>
  （24）
  <br/>
  11岁女儿：（在观看911事件的纪录片）
  <br/>
  我：“当它发生时，我在上高中。”
  <br/>
  11岁女儿：“哇。你像历史一样老。”
  <br/>
  （我现在也有这种感觉。）
 </p>
