 <p>
  <strong>
   本人愿意退出团队组织，并且拒绝入党
  </strong>
  <br/>
  本人生在山东，长在山东，现在已经到了而立之年心中还是记着从小我的爷爷的教育：“做人不能伤天理”。到现在我也是尽力做到这一点。
  <br/>
  但从06年中专毕业参加工作总是感到身边太多的人总是与自己不合拍，处处充满了尔虞我诈，人际间的关系大都是钱钱钱，感觉童年的童真和互相的信任再也找不回来。我也在考虑为什么这样。
  <br/>
  以前我没看外面的新闻的时候看的是墙内的网易新闻（2018年前的网易新闻还是能看到一些比较正统的新闻和网友评论，这也使得网易遭受到太多的整治……）开始进一步了解了墙内的这些情况。也经常转发一些社会面的新闻（比如官员贪腐；环境污染；校园欺凌等等）到自己的微信朋友圈。但得到的大都是嘲笑和类似“不要关心政治，过好自己的小日子”这种回复。其实自己已经差不多明白了：墙内这种制度本身就是不合理的，因此自己在工作中婉言拒绝了领导的多次“入党”要求，那时候心中对这个词真的不再有好感。
  <br/>
  到了15年，我在一个偶然的机会捡到了有缘人挨家挨户赠送的小光碟，里面是自由门和无界等翻墙软体还有一些传法的资讯资料，这使得我有翻出墙外看外面新闻的机会。虽然后期已经购买了收费版的机场服务，但这些软体一直保留在我的电脑中备用。
  <br/>
  看了6年多的外网新闻，使我了解到了太多墙内被封杀的内容。我也在这段时间经常在墙内的互联网上发表一些自己的看法，但得到的却是微博号被封了3个；哔哩哔哩号被封了1个。因为我现在自身在墙内，真心不敢继续做出再进一步的做法，也了解到这个组织对人的监控和处置方式。我只能做到不与这个组织妥协和尽量不再做出过于“出格”而被封账号的行为。但是，我还是保持的我最初的信念：做人不能伤天理，用自己的良心做事就够了。
  <br/>
  我目前还没有宗教信仰，但我也绝不相信这个组织说的：“人是猴变的”这种说法。
  <br/>
  我在生活中也经常和身边的朋友分享我看到的这些来自墙外的节目，虽然得到了太多类似于反动；假消息；再看这些举报你……等等回复，我也只能一笑了之。不过也不是完全没有效果，我的朋友已经有5个左右的已经再看墙外的新闻了。
  <br/>
  我前些年曾经用化名来贵站提交过一次声明，但后期好像忘记了找不到了。今天我想重新提交一次声明：本人退出曾经加入过的少先队、共青团 ，以前拒绝过加入党员，以后也会坚持拒绝加入。请原谅我的懦弱，不敢用过分的形容词形容这个组织。没办法，我还没有出墙，我的父母也没有出墙…..我想先做一个旁观者和时间的见证者，看到这个组织被天灭的时候。也在这里向上天祈求我和我的父母生活平安，因为他们心里并不坏，只是被这个组织迷洗的时间太长了，心里的可能需要太长的时间。
  <br/>
  我在这里感谢youtube上的石涛先生的节目，给予我在这里写出这篇文字的勇气。也感谢新唐人电视台的《中国禁闻》让我了解了国内真正的情况。还有文昭；小民；建民……等等一些的播主提供的有价值资讯。

  <strong>
   退出少先队、红卫兵、中共共青团
  </strong>
  <br/>
  我们从一出生开始，就是在阴霾中成长。那个时候我们对未来满怀期望，因为我们被教育的自己认为是这个国家的主人！从小父亲被批斗，与我们分离，我们没有父爱；在被鄙夷的眼光中战战兢兢小心翼翼的活着。那个时候只有入少先队、红卫兵、共青团才是人们眼中的好孩子，我们为了做个好孩子什么苦都吃，但是都得不到认同；都是到了大家基本上全部加入了，才有我们的名额。小小的年纪我在思考：难道我都不如那些打架骂人的同学吗？从小我连一句脏话都没有说过，我规范着自己的言行！游离在他们的团伙之外，用眼睛和思想看着我们生活的周遭。当共青团让我加入时，我的心彻底对未来绝望了。我知道善良直率在我们生活中是有罪的！我们只是个奴隶！是比在监狱中的犯人多了些可以走动的地方而已。其他的人的选择我们是没有的！到处充斥着仇恨、怨恨、鄙视、谎言、尔虞我诈；没有一点信任、信誉、爱！一个被领导成这个样子的社会可以永续吗？一个这样的政党和政权可以永续吗？这个阴霾太久了！100年，是我们三代人的噩梦，摧毁了唯人的底线！他们要毁灭人性让人变成奴性！共产党是人类共同的敌人！当人们不在友爱和有爱时，上天要我们干什么？我相信举头三尺有神明！我们期盼上天的恩泽能让我们生存的大地重现阳光！我们从我们自己做起，从现在做起我们以我们的生命严重声明退出少先队、红卫兵、中共共青团这个邪恶的组织！
