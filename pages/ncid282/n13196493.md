  <br/>
  近些年，中共对互联网的严控可以说是前所未有的，进各大网吧，都要实名登记，部分网吧，还要求扫脸！办手机卡时，也要先实名登记，再进行活体认证，才能激活。
  <br/>
  不过，共产党像这样强制推广实名制，并没有减少社会上的诈骗事件。我所在的地区，几乎每天都有电信诈骗案例，数额小的，被骗了几千元，数额大的，一次性被骗好几十万元的都有。以前骗子打电话时，还会问这问那，现在骗子打电话来时，直接把人家的姓名、地址、年龄以及家庭成员等信息全部掌握了。
  <br/>
  其实，被中共祸害下的社会，人们是毫无道德修养的，越来越多的女性，利用网络交友、网恋等，骗别人的金钱和感情，本人就遇到过多起这样的事件。而且中共利益集团，跟国内这些毫无良心的资本家们，每年都会炒作“情人节”、5月20日、七夕节、以及“11月11日光棍节”，来诱导无脑的男同胞为一些渣女消费，成为被收割的“韭菜”！
  <br/>
  最近两三年间，本人在网络上遇到过很多女生，她们一开始就说要跟我搞对象，然后发一些经过重度美颜的照片。凡是个正常男生，只要见到美女，那多多少少是有些心动的，聊着聊着，对方就提出来要见面。但是见了面之后，我感觉她们真人跟美颜照片都有不小的差别，不太像同一个人，但是见面回去了之后，对方经常像我索要红包和礼物，我考虑到已经交往过一段时间，就给对方发了些红包。但是没过多久，对方找各种莫名其妙的理由说想放弃这段感情，我立马感觉自己被骗了！后来，我发现这些女生，用不同的社交账号，同时在跟多名男性交往，在中国大陆俗称“养鱼塘”！而且，不少女生，还以此为荣！
  <br/>
  中共已经来到中国大陆祸害了100年。当年，中华民国在大陆时，百姓大部分民风淳朴，女性也受过很多良好的教育，人们还是有很多道德修养的。但是自从1949年中共篡夺政权后，这片土地上再也没有之前的味道了，到了文革时期，流氓地痞横行，把整个社会糟蹋的近乎崩溃！改革开放之后，虽然人们大部分吃饱了饭，但是骗子横行，很多女性的道德修养已经丧失殆尽！人们都向钱看，为了钱，什么样的缺德事都能做出来。
  <br/>
  这几年，中共搞的反腐败运动，也是一个彻彻底底的骗局，打着反腐的幌子，来干掉政敌，从而想为这个腐臭的中国共产党，挽回一些执政的颜面，这就是共产党“既要当婊子，还要立牌坊”的丑陋一面。
  <br/>
  曾在小学和初中时，被人忽悠，盲目加入了中共的少先队和共青团，现在醒悟了，觉得当时的自己很天真。现在决定退出中共的团和队，并与这个流氓组织说拜拜！
