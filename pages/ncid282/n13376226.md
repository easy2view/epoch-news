 <p>
  <strong>
   退出中共的一切邪恶组织
  </strong>
  <br/>
  我父亲李明亮是一名军人，参加过抗美援朝，也经历了文化大革命的迫害。在世时，每次他喝了酒以后，总给我们讲那些惨烈的故事。
  <br/>
  先说抗美援朝父亲亲眼所见的场景。那时上战场没有经验，上去简直就是送死，人们都充当了炮灰，一排人冲上去，倒下了；又一排冲上去，又都倒下了。重伤患没有人给治疗，就是自生自灭，轻伤患就靠自己的运气了。
  <br/>
  又讲了文化大革命令人恐怖的情景。我父亲上班去就被关在小屋里写交代材料，之后游街，用白纸糊了三个大高帽子，我亲眼所见，最大的一个三米高，上面写着“打倒某某某走资派”、“永世不得翻身”。后来动私刑，用一根大粗绳子穿在房梁上，一头拴在父亲倒背的手上，另一头在施刑人的手里。这时施刑人用力拽绳子，一直把人拽到房顶，而后猛一撒手，人迅速下降，摔在地上。这叫“坐飞机”。父亲当时都五十多岁了，被迫害的生不如死。
  <br/>
  我父亲还讲了一个和他一起被批斗的人的故事，这个人说了一句辩解的话，就被批斗，军代表说：“这个人反对毛主席的文化大革命，是黑五类反动派！”问一红卫兵：“他该不该死？”人们起哄说：“该死！”军代表又问：“该不该杀？”，这帮人又起哄说：“该杀！”，几个红卫兵从库房里拿来了铁锤，红卫兵们一拥而上把这个人按倒，用砖头垫在他的脚踝骨下面，用锤子一阵乱砸，那个人的脚踝骨变成了肉泥。当时吓得父亲大气不敢出，晚上在小黑屋里哆嗦了半宿。老郑是火房里做饭的师傅，给父亲送饭时，悄悄告诉父亲说：“老李，你赶快逃命吧，今晚要除掉你，我给他们端菜时听到的。”说完锁上门就走了。父亲拼了命似的把小窗户上的木板子拽下来，跳窗逃走了，总算保住了一条命。
  <br/>
  一九九八年，父亲经人介绍说法轮功祛病健身有奇效。他炼功不到一个月，身上的枪伤、弹伤、血压高等毛病都恢复了正常，腿不疼了，走路也不拿拐棍了，见人就说：“法轮功真好，快炼吧！”到处去洪法。一九九九年邪党迫害法轮功，父亲单位让他写保证书，并威胁说不写就抓起来送劳教所，停发工资。每月父亲去开支，邪党书记都问：“老李，还炼吗？再炼，摘了你的鸟食罐子，别吃饭了！”父亲在家说，文革又来了，运动又来了，吓的他精神恍惚，精神受到了极大的刺激，经常想起文革时受的那些迫害，一有人敲门就吓的不行。
  <br/>
  二零零一年冬，父亲摔倒在卫生间里，再也没起来。现在我为我那冤死的老父亲发表声明：彻底退出中共邪党的党、团、队、团长、连长、班长等一切邪恶的组织，做一个清白的人。以此来安慰我父亲的在天之灵。
