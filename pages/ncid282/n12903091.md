  <strong>
   声明退出共产党的组织
  </strong>
  <br/>
  我今天内心强烈地声明退出共青团员，小时候觉得作为一个团员是多么地光荣，历史课本上也讲述着共产党是多么地伟大，法轮功是多么地可怕，可这一切不过是邪党造谣抹黑罢了，真正可怕的是他们。
  <br/>
  他们是刽子手，他们对法轮功学员残害、把香港变得不再民主、压迫新疆劳动人民、用坦克碾压8964学生、甚至还把病毒放出来危害全球人民，这哪一件是人干的事情，我为自己是共青团员赶到羞耻，还好我在大学了解到这些之后，没有申请入党积极分子。
  <br/>
  邪党最擅长洗脑抹黑。他们已经把我母亲洗脑的很严重了，我把这些告诉我妈时，她不信而且和我吵架，她说的话在我看来丝毫没有逻辑性，可她却觉得自己说的和真的似的，真的是可怕。我也不知道她是装不知道还是不敢相信这些。
  <br/>
  我也感激和钦佩那些敢于站出来揭露共党邪恶的人，特别是那些颇有名望的人，这得顶着多大的压力，为他们正义的行为点赞。
  <br/>
  愿上帝可以让更多的墙内人看到真相，早日了解到邪党的恶。中国现在还是“毛”病不改，恶“习”难除的局面，希望邪党早日被推翻，还中国人民一片净土。
