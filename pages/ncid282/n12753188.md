 <p>
  <strong>
   谈谈一年来的经过感想
  </strong>
  <br/>
  我是去年高考，恰逢疫情出现，学校停课，小区把人像狗一样地关着。出入要听从那群凶神恶煞的走狗——中共党员。
  <br/>
  那时老师还不忘盘剥，明明在家上网课，只是不耐烦的应付。只见她用手机照着试卷一道题，屏幕上大大的光影，明明什么都看不见，直播间的同学们却像东郭先生一样应和着。
  <br/>
  我常常思考这不是人性与生俱来的虚伪，这是体制带来的荼毒！平常只有晚课收费，疫情期间，网课她什么都不讲还要一节课八十想！考好大学，想有好成绩，别做梦了！厉害国的屁民不敢奢望，只能拿着华为看战狼，歌颂我党的丰功伟绩！
  <br/>
  后来的我于去年八月考上了一所二本大学的计算机专业，充满奴性的父母于是感激涕零，叩谢隆恩的架势对我说：“我们家都是出苦力的命，终于有你一个大学生，不但光宗耀祖，当个公务员，祖上积了德了，都要感谢党和政府，老师啥的”。他们无法接触真相，我无法也不愿意，更不能向他们解释，顺着说就得了。
  <br/>
  他们不会知道，这个政府严格封锁消息，强大的公关战；
  <br/>
  他们不会知道，每个人被刻上了正确的人类集体记忆；
  <br/>
  他们不会知道，这个世界的悲哀在于死人不能说话，活人同样不能；
  <br/>
  太多的无奈压在身上，大学我刚上一个学期，压抑的几乎喘不上气。我厌倦了尔虞我诈、谎话连篇的日子。保全自己，我不敢乱说一言。虽然，大学里每周的团会、青年大学习、政治课不能不上。但我真的很想和这个恶党划清界限，庆幸自己在这个无助、污浊、谎言弥漫的地狱里没被洗脑。假以时日，有机会一定出国，逃离中共的魔爪。在为了这个红色中国想想出路！
  <br/>
  感谢大纪元，感谢自由门，感谢为正义而遭迫害的人们！我写着写着就哭了! 我声明退出中共一切组织！
