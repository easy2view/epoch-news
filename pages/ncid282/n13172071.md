  <strong>
   退出少先队、共青团等中国共产党的一切邪教组织
  </strong>
  <br/>
  我是一个来自农村的00后女孩！小时候很快乐，但是随着年龄的增加，懂得的道理也越来越多！
  <br/>
  在上小学和初中期间，学校通过各种方式让我们加入少先队和共青团，我们那时啥也不懂，就盲目加入了！
  <br/>
  初中毕业后，我来到了一所幼师学校。但是来到这个学校后，时不时要参加校团委举办的各种活动。我是带着梦想来学习的，可总感觉像加入了一个传销诈骗组织。
  <br/>
  老师总是告诉我们要爱党爱国，还动不动让我们唱红歌，跳革命舞蹈。学校里的学生会干部很多都是团员，她们蛮横霸道，专门欺负我们这些新来的学生，还私自扣我们的零花钱！
  <br/>
  我们每晚上自习时，必须要看新闻联播。但我发现我的生活，跟新闻联播上说的很不一样。我家里比较贫穷，然而新闻上动不动就说我们的生活很幸福，外国人都生活在水深火热之中，这很明显是骗人的谎话！
  <br/>
  上个月，我们河南发生了严重的洪灾，死了很多人，但是在新闻联播上却看不到任何关于这件事的报道，更不会说死了多少人，那么多鲜活的生命就这样消失了。可是美国只要有个枪击事件，新闻联播绝对是第一时间报道。
  <br/>
  突然感觉，从前上学时，被学校欺骗，也可以说是被共产党欺骗了！我感到有苦说不出，家人辛苦挣钱供我读书，但他们却不知我正在被一个邪教组织毁了人生！这个邪教组织就是共产党。
  <br/>
  小时候听人说共产党挺坏的，我当时不信，甚至还想怼两句回去，但是现在全信了！所以本人从今天起决定退出少先队、共青团等中国共产党的一切邪教组织！

  </strong>
