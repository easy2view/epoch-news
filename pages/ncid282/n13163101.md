  </strong>
  <br/>
  本人从小出生在大陆沦陷区的农村，那个时候家里很贫穷。小时候读书，好像被忽悠加入了什么少先队。那个时候什么都不懂，看着别人加入，我也就加入了！
  <br/>
  后来我成年了，结了婚，成了家，有了几个孩子。但是我无时无刻不感到生活压力太大！之前在老家跟着弟兄做点小生意，但是特别不景气，倒闭了！我又被迫跟妻子去江苏打工。家里的几个孩子都由我父亲照看！但是我的老父亲平时喜欢赌博，孩子的生活没人照管，后来难免都学坏了！在外面打工也不是很景气，去年我的手腕还出现了损伤。于是我想回老家干！
  <br/>
  我平时看到共产党搞的宣传非常好，特别是最近些年的“扶贫政策”，还说要在2020年全面脱贫。但我发现，共产党的一群贪官都是流氓，他们嘴上说的东西都十分动听，但是真正做到的几乎没有。我们家一直到现在都是十分贫穷的。
  <br/>
  无奈之下，我和妻子商量，决定今年春节后，自己经营一家小餐馆。但是门店开了之后，共产党的那些所谓的执法人员经常来店里骚扰，他们总是以检查为理由，索要一些费用！我的门店本来生意就不好，赚点钱不容易，还要给这群穿着“狗皮”的人员送钱！
  <br/>
  我实在感到生活在大陆沦陷区，压力很大，人每天活着都感觉不到快乐！这一切都是因为那个万恶的共产党害的！共产党不除，我们这些底层的老百姓永远都不可能有生活的希望！
  <br/>
  本人声明从此退出共产党的一切组织，不管我以前加入过什么，总之这些大大小小的党组织现在跟我没有一点关系了！
