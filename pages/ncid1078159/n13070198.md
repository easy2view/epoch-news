 <p>
  【大纪元2021年07月06日讯】（大纪元记者林燕综合报导）
  <span class="s3">
   中美关系紧张之际，中国网上出租车平台“
   <ok href="https://www.epochtimes.com/gb/tag/%E6%BB%B4%E6%BB%B4%E5%87%BA%E8%A1%8C.html">
    滴滴出行
   </ok>
   ”寻求在美上市
  </span>
  <span class="s4">
   。跟去年蚂蚁金服上市前几天被叫停不同，
   <ok href="https://www.epochtimes.com/gb/tag/%E6%BB%B4%E6%BB%B4%E5%87%BA%E8%A1%8C.html">
    滴滴出行
   </ok>
   才上市几天就遭到中共接连重击。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   《华尔街日报》周一（7月5日）报导说，据知情人士透露，滴滴出行在美国上市前几周，中共网络安全监督机构建议，让它推
  </span>
  <span class="s6">
   迟
  </span>
  <span class="s1">
   首次公开募股（IPO），以及对其网络安全进行彻底的自我检查。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   但滴滴出行在当局没有直接下令叫停IPO的情况下，选择了继续推进。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   《华日》引述熟悉讨论情况的人士消息说，滴滴出行从不同监管机构收到不同的信号。
  </span>
 </p>
 <p class="p1">
  <span class="s9">
   报导还说，
  </span>
  <span class="s1">
   上述消息人士表示，中共官员们，尤其是网络管理的官员，认为
  </span>
  <span class="s1">
   在美国上市会面临公开更多信息。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   随后，中共当局对滴滴出行祭出的监管行动让投资者感到惊讶，因为这些行动是在公司上市后几天就接连发生，外媒报导说，此举会打击中国公司走向世界的步伐。
  </span>
 </p>
 <p>
  纽约时事评论员朱明表示，在预期美国要求中企满足其它企业的既定要求，遵守美国上市公司规定的情况下，中共当局已愈发跋扈，要求中企必须遵从党的法律，即便已经在境外上市，也必须为不服从付出惨痛代价。
 </p>
 <p>
  朱明说，中共当局此举释放的另外一个信号是，即使可能对中企争取国际投资者的能力构成威胁，中共也决定要插手。
 </p>
 <h4 class="p1">
  保护敏感数据是中共打击滴滴的噱头
 </h4>
 <p class="p9">
  <span class="s7">
   中共并不保护中国的敏感数据和网络，中国是全球网络审查最严苛的国家。
  </span>
 </p>
 <p class="p9">
  <span class="s10">
   总部设在华盛顿的国际人权组织自由之家（
  </span>
  <span class="s11">
   Freedom House
  </span>
  <span class="s10">
   ）
  </span>
  <span class="s11">
   2020
  </span>
  <span class="s10">
   年发布的网络自由报告，在
  </span>
  <span class="s11">
   65
  </span>
  <span class="s10">
   个接受调查的国家中，中国排名垫底
  </span>
  <span class="s12">
   ，排在俄罗斯、古巴、伊朗以及叙利亚之后。
  </span>
 </p>
 <p class="p10">
  <span class="s10">
   自由之家的报告说，没有哪个地方的审查制度比中国更严格、更系统。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   纽约时事评论员朱明说，过去曝光的多个人权迫害案例都显示，中国私人科技公司和国有公司一直跟中共公安部共享数据和合作。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   “中共声称的保护敏感数据和网络只是用来维持统治、迫害异议人士和宗教信仰团体，”朱明说，“重敲滴滴不是为了保护数据，这是中共的一种姿态，是为挑衅以及打破美国监管体制试水，同时也是中共惧怕权力被撕开一个口子后崩塌。”
  </span>
 </p>
 <h4 class="p1">
  中共害怕中企守美国规矩 加强数据控制
 </h4>
 <p class="p1">
  <span class="s1">
   《华日》引述熟悉此事的人士称，中共网络安全当局特别关注的是美国国会通过的一项标准要求，即未来的上市公司必须向美国证券交易委员会披露“重大合同”，或有关公司的主要供应商和供货商信息。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   上述人士表示，尽管滴滴出行等公司将其用户和交通流量数据存储在中国境内的服务器上，但网络空间管理局的官员担心，这些服务器的设备如果从国外采购，可能有中共认为的安全漏洞。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   分析人士称，中共当局对滴滴出行的审查过程可能持续数月，涉及约十几个政府机构，包括公安部和发改委。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   美国上市公司会计监督委员会确定，有二百多家在美国上市的外国公司——其中大部分总部设在中国——这些公司的审计工作都不让美国机构检查。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   国会联邦参议员卢比奥（Marco Rubio）6月呼吁，美国证券交易委员会阻止滴滴上市，除非它采用与其它美国上市公司相同的审计监督标准。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   一直以来，中共政府引用旨在保护国家安全和国家机密的法律，来阻止美国监管机构审查中国公司的审计文件。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   2020年，中共更开始阻止中国公司与海外监管机构合作，或在没有政府授权的情况下交出文件。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   自去年年底以来，习近平已加强对技术部门及控制数字数据存储作为监管关键领域。
  </span>
  <span class="s1">
   他亲自出面叫停了阿里巴巴旗下的金融科技公司蚂蚁金服的上市计划，就在蚂蚁金服计划11月上市的前几天。
  </span>
 </p>
 <h4 class="p1">
  滴滴受挫事件回顾
 </h4>
 <p class="p1">
  <span class="s1">
   滴滴出行6月30日在纽约证券交易所上市，共筹集约44亿美元，成为自阿里巴巴集团控股有限公司以来中国公司最大金额的一次公开募股。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   7月1日，中共当局举行盛大的百年党庆活动。
  </span>
 </p>
 <p class="p5">
  <span class="s7">
   7月2日，
  </span>
  <span class="s1">
   中共国家互联网信息办公室宣布，为防范国家数据安全风险，已经启动对
  </span>
  <span class="s8">
   “
  </span>
  <span class="s1">
   滴滴出行”
  </span>
  <span class="s1">
   的审查。
  </span>
 </p>
 <p class="p5">
  <span class="s1">
   7月3日，
  </span>
  <span class="s8">
   “
  </span>
  <span class="s1">
   滴滴出行
  </span>
  <span class="s8">
   ”
  </span>
  <span class="s1">
   的全球公司副总裁李敏回应网络传闻，称该公司将中国所有用户和道路数据存储在其国内的服务器上，
  </span>
  <span class="s8">
   “
  </span>
  <span class="s1">
   绝对不可能
  </span>
  <span class="s8">
   ”
  </span>
  <span class="s1">
   将数据泄露给美国。
  </span>
 </p>
 <p class="p5">
  <span class="s1">
   7月4日，中共国家互联网信息办公室发布通告称，滴滴出行
  </span>
  <span class="s8">
   “
  </span>
  <span class="s1">
   严重违法违规收集使用个人信息
  </span>
  <span class="s8">
   ”
  </span>
  <span class="s1">
   ，要求下架滴滴出行
  </span>
  <span class="s8">
   App
  </span>
  <span class="s1">
   。
  </span>
 </p>
