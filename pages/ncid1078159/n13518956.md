   </ok>
   能力应对中国大陆日益增长的恐吓。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   这些不愿透露姓名的官员说，他们还没有就如何加快交付Block 70
  </span>
  <span class="s2">
   F-16
  </span>
  <span class="s1">
   战斗机提出解决方案。这些飞机目前计划在
  </span>
  <span class="s2">
   2026
  </span>
  <span class="s1">
   年底前交付，由洛克希德
  </span>
  <span class="s1">
   ‧马丁公司制造，飞机配备一些新功能。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   一位台湾高级官员告诉路透社，台湾政府已经私下向美国政府表达了希望更快交付的愿望，因为
   <ok href="https://www.epochtimes.com/gb/tag/%E5%8F%B0%E6%B9%BE%E7%A9%BA%E5%86%9B.html">
    台湾空军
   </ok>
   正在用战机拦截中国大陆越来越多的侵略性军事飞行。
  </span>
 </p>
 <p class="p1">
  <span class="s1">

 <p class="p1">
  <span class="s1">
   “都是风险评估
  </span>
  <span class="s2">
   ……
  </span>
  <span class="s1">

  </span>
  <span class="s2">
   F-16
  </span>
  <span class="s1">
   被认为是一种高度机动飞机，将投入在空对空作战和空对地攻击活动。
  </span>
 </p>
 <p class="p1">
  <span class="s1">

  </span>
  <span class="s2">

  </span>
  <span class="s2">
   66
  </span>
  <span class="s1">
   架新造
  </span>
  <span class="s2">
   F-16 C/D Block 70
  </span>
  <span class="s1">

  </span>
  <span class="s2">
   F-16
  </span>
  <span class="s1">
   机队的地区之一。台湾的
  </span>
  <span class="s2">
   F-16
  </span>
  <span class="s1">
   飞机总数，包括旧版战机，在
  </span>
  <span class="s2">
   2026
  </span>
  <span class="s1">
   年之前达到
  </span>
  <span class="s2">
   二百
  </span>
  <span class="s1">
   多架。
  </span>
 </p>
 <p class="p1">
  <span class="s1">

  </span>
 </p>
 <p class="p1">
  <span class="s1">
   目前，洛克希德
  </span>
  <span class="s1">
   ‧马丁公司在南卡罗来纳的新
  </span>
  <span class="s2">
   F-16
  </span>
  <span class="s1">

 </p>

  <span class="s1">

  <span class="s1">
   ‧马丁公司为台湾的
  </span>
  <span class="s2">
   F-16
  </span>
  <span class="s1">
   战斗机改变交付时间表。
  </span>
 </p>
 <p class="p1">
  <span class="s1">
   该人士拒绝猜测，即使决定加速交付，台湾能在多长时间内获得新造的
  </span>
  <span class="s2">
   F-16
  </span>
  <span class="s1">
   战斗机。任何这样的努力都会因生产限制而变得复
