 <p>
  【大纪元2021年06月18日讯】（大纪元记者林燕综合报导）白宫周四（6月17日）表示，可能即将举行美国总统拜登和中共总书记习近平之间的直接会谈，这可能是美、中国家元首之间的首次会晤。随后美国国务院对此事进行了解读。
 </p>
 <p class="p3">
  美国国家安全顾问杰克‧沙利文（
  <span class="s2">
   Jake Sullivan
  </span>
  ）周四告诉媒体：
  <span class="s2">
   “
  </span>
  拜登总统将在未来一个月以某种方式与习主席联系，以评估我们的关系，并确保我们进行类似昨天与普京总统的有意义的直接沟通，我们正致力于此。”
 </p>
 <p class="p3">
  他补充说：
  <span class="s2">
   “
  </span>
  现在只是一个什么时间以及用什么方式接触的问题。”
 </p>
 <p class="p3">
  美中作为世界两个最大的经济体，正在经历越发下行的激烈碰撞关系。而且拜登刚结束欧盟的一周访问，强化盟友关系以及跟竞争对手俄罗斯总统普京会晤。
 </p>
 <p class="p3">
  自上任以来，拜登已经和盟友一起谴责和惩罚北京在新疆地区涉嫌侵犯人权，镇压香港民主，以及在南中国海的剑拔弩张行为。
 </p>
 <p class="p3">
  中共官员一贯用美国干预中国内政进行回击，对迫害事件依然不断否认。
 </p>
 <h4 class="p3">
  国务院：会面建议不意味着
  <ok href="https://www.epochtimes.com/gb/tag/%E4%B8%AD%E7%BE%8E%E5%85%B3%E7%B3%BB.html">
   中美关系
  </ok>
  有突破
 </h4>
 <p class="p3">
  不过，在沙利文发表声明后不久，美国国务院告诉媒体，会面建议不标志着美中之间关系有任何突破，这只是拜登致力于外交的体现。
 </p>
 <p class="p3">
  国务院发言人内德‧普赖斯（Ned Price）说，沙利文“是在说总统提出的主张，即个人外交是不可替代”。
 </p>
 <p class="p3">
  <span class="s2">
   “
  </span>
  他指出了这点，这并不是拜登总统和普京总统会晤独有的。当涉及到我们的原则性外交时，这一点全面适用。
  <span class="s2">
   ”
  </span>
  普赖斯解释说。
 </p>
 <p class="p3">
  他还补充说：“目前没有什么可以预告，但如果条件合适，情况需要，我们会（对
  <ok href="https://www.epochtimes.com/gb/tag/%E6%8B%9C%E7%99%BB%E4%B9%A0%E8%BF%91%E5%B9%B3%E4%BC%9A%E6%99%A4.html">
   拜登习近平会晤
  </ok>
  ）保持开放的态度。
  <span class="s2">
   ”
  </span>
 </p>
 <h4 class="p3">
  何时可能会晤 白宫列出多个选项
 </h4>
 <p class="p3">
  至于何时可能会晤，国家安全顾问沙利文周四透露，拜登
  <span class="s2">
   “
  </span>
  将寻找机会与习主席继续保持联系
  <span class="s2">
   ”
  </span>
  。
 </p>
 <p class="p3">
  <span class="s2">
   “
  </span>
  很快我们（双方）就会坐下来，为两位总统的接触制定合适的方式。”他说，
  <span class="s2">
   “
  </span>
  可能是电话，可能是另一个国际峰会期间进行会晤，也可能是其它方式。”
 </p>
 <p class="p3">
  苏利文说，
  <span class="s2">
   10
  </span>
  月，拜登和习近平预计出席在意大利举行的
  <span class="s2">
   G20
  </span>
  峰会，这是两位领导人举行峰会的潜在地点，不过他补充说，
  <span class="s2">
   “
  </span>
  我们目前没有任何具体计划
  <span class="s2">
   ”
  </span>
  。
 </p>
 <p class="p3">
  习近平两年前在日本大阪
  <span class="s2">
   G20
  </span>
  峰会期间，曾与时任美国总统唐纳德‧川普（特朗普）会晤。
 </p>
 <h4 class="p3">
  拜登：跟习近平不是老朋友 纯属业务关系
 </h4>
 <p class="p3">
  周三（16日），拜登在日内瓦会见普京。美俄关系因俄罗斯镇压抗议者，以及华盛顿对网络攻击和干涉选举指控而急剧降温。
 </p>
 <p class="p3">
  会晤结束后，一名记者提问拜登，他是否有计划与习近平进行
  <span class="s2">
   “
  </span>
  老朋友之间的对话
  <span class="s2">
   ”
  </span>
  ，讨论中方在调查冠状病毒（中共病毒）大流行起源上的合作。
 </p>
 <p class="p3">
  拜登接过话说：
  <span class="s2">
   “
  </span>
  让我们把话说清楚，我们（指他和习近平）彼此熟知，但我们不是老朋友。只是纯粹的业务关系。
  <span class="s2">
   ”
  </span>
 </p>
 <p class="p3">
  拜登和习近平在过去的职业生涯中曾多次会面。在习近平升任中共最高领导人之前，跟拜登同时期担任过副手。
 </p>
 <p class="p5">
  <span class="s3">
   拜登上任后，他们在
  </span>
  <span class="s2">
   2
  </span>
  <span class="s3">
   月进行了首次电话交谈，习近平在
  </span>
  <span class="s2">
   4
  </span>
  <span class="s3">
   月在线
  </span>
  参加了拜登在白宫举行的气候变化峰会，但他们还没有以国家最高领导人身份进行面对面的会谈。
 </p>
 <p class="p3">
  拜登在
  <span class="s2">
   3
  </span>
  月说，习近平骨子里没有民主。
 </p>
